# AI Data Chat System

## 📌 Project Description
The AI Data Chat System allows users to interact with sales data using simple English questions.  
Instead of writing SQL queries, users can ask questions like "Show total sales" or "Which product sold the most?" and get instant results.

---

## ❗ Problem Statement
Many people do not know SQL or technical tools to analyze data.  
They depend on technical experts even for simple data queries, which wastes time and effort.

---

## 💡 Solution
This project uses an AI-based approach to convert user questions into database queries.  
It makes data analysis simple, fast, and user-friendly.

---

## 🚀 Features
- Chat-based interface for easy interaction  
- Converts natural language to SQL queries  
- Fetches data from database instantly  
- Displays results clearly  
- Simple and beginner-friendly system  

---

## 🛠 Technologies Used
- Python  
- SQLite  
- HTML (for chat UI)  
- Pandas  
- Matplotlib (for graphs)

---

## ▶️ How to Run the Project

1. Download or clone the repository:
   git clone https://github.com/your-username/AI-Data-Chat-System.git

2. Open the project folder

3. Install required libraries:
   pip install pandas matplotlib

4. Run the database file:
   python create_db.py

5. Run the main program:
   python main.py

6. Open chat_ui.html in browser
